# Pruebas técnicas – ECM/BPM (OnBase / Alfresco)

Este repositorio contiene un conjunto de pruebas técnicas para evaluar candidatos a:

- Practicante en Implementación de Proyectos ECM/BPM

Las pruebas se enfocan en:

- Modelado de procesos (BPMN) y modelos de datos
- Programación en C# sobre un modelo simulado de OnBase
- Programación en Java sobre un modelo simulado de integración con Alfresco

---

## Contenido

- `prueba1-diagramas/`  
  Prueba de **análisis y modelado**: historias de usuario, diagrama BPMN, diagrama de clases,
  supuestos y preguntas al área de negocio.

- `prueba2-onbase-csharp/`  
  Prueba de **programación en C#** sobre un modelo simulado de OnBase.  
  El candidato debe implementar la lógica para generar y asignar números de radicado.

- `prueba3-alfresco-java/`  
  Prueba de **programación en Java** para un cliente simulado de Alfresco.  
  El candidato debe implementar la lógica de carga de documentos y manejo de metadatos.

Cada carpeta incluye su propio `README.md` con los detalles, objetivos y forma de entrega.
